# HLS Video Downloader

React приложение для скачивания HLS видео из .m3u8 манифестов.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Использование

1. Вставьте URL .m3u8 манифеста в поле ввода
2. Нажмите кнопку "Скачать"
3. Дождитесь завершения скачивания всех сегментов
4. Видео автоматически скачается в формате .mp4

## Особенности

- Парсинг m3u8 манифеста
- Скачивание всех сегментов видео
- Объединение сегментов в один файл
- Отображение прогресса скачивания
- Автоматическое скачивание готового файла
